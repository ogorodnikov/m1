Tutorial

Quantum computers use properties of quantum states to perform calculations.
Quantum information is stored and processed using quantum bits – qubits.
Quantum algorithms for certain problems can be significantly faster than classical algorithms.

M1 Project proposes to run quantum algorithms on simulator or real quantum computers – kindly provided by IBM Quantum and Qiskit Community.

How to use M1:

  1. Choose Algorithm from the list
  2. Check Wikipedia info, Qiskit explanation and M1 code
  3. Open Algorithm page
  4. Set parameters
  5. Run algorithm on Simulator
  6. Check task Logs and Results
  7. Run algorithm on Real Quantum Computer
  8. Check task Logs and Results as well
